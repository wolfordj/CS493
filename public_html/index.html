CS493
  • Syllabus – This is the course syllabus. It contains important course information from including things like grading and contact policies.
  • Course introduction, cloud introduction and REST introduction – This module is designed to give you a basic outline for the course and to give you an idea about why we care about cloud computing and what this whole REST thing is about.
  • REST API Usage – This module should introduce you to the concept of using a REST API which will likely be different from the APIs you may have made or interacted with in the past.
  • REST API Creation – Having spent some time interacting with a REST API this module will introduce the concept of making a REST API that others can interact with.
  • Yet More REST API Creation – Making REST APIs can be a bit tricky. So this module expands upon the prior giving you some more time to get this built.
  • Advanced REST API Concepts – Having made a generally functional REST API this will go into some more complex concepts like paging and caching.
  • Some Bits of Security – Security in the cloud is a HUGE topic. This will cover authorization and OAuth 2.0, a tiny sliver of security, but an important sliver to be able to play well with others.
  • More Authorization and Authentication – This week goes into a bit more detail on the topics of authorization and authentication as well as giving a broad look at security.
  • Docker – Docker is a tool that is used to create preconfigured systems that can be quickly created and destroyed with little overhead.
  • Non-Relational Databases – This week goes over the concept of non-relational databases. They have some limitations and abilities that are rather different from a typical relational database.
  • Wrap-up – This module gives you some time to complete the final project while offering a bit of parting wisdom in the way of what is coming on the horizon and how to test some of your applications.
